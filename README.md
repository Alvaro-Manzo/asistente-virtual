# Asistente Virtual Multifunción en Python

Este proyecto es un asistente virtual que escucha tu voz y responde de forma inteligente y personalizada. Puede buscar información, darte frases motivacionales, mostrar datos curiosos, manejar precios de criptomonedas y acciones, reproducir videos, contar chistes, calcular divisas ¡y mucho más!

## Características principales

- **Reconocimiento de voz** (speech to text) y síntesis de voz.
- Frases motivacionales y palabras del día.
- Datos curiosos random y chistes.
- Consulta de precios de criptomonedas, metales preciosos y acciones populares.
- Apertura automática de sitios útiles (YouTube, Google, Wikipedia…).
- Comandos secretos y “easter eggs” divertidos.
- Ejecución de cálculos, conversiones simples y mucho humor.
- Listo para expandir más comandos o conectarse a nuevos servicios.

## Requisitos

- Python 3.8 o superior.
- Micrófono funcionando.
- Conexión a internet (para funciones como Wikipedia, precios, chistes online, etc).

## Dependencias principales
